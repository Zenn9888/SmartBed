簽床系統操作手冊

步驟 1：打開資料夾

找到 SmartBed 資料夾，並點選啟動程序.bat。若無法執行，請進入 src 子資料夾，點選 啟動程序.bat，系統將自動安裝所需環境。

步驟 2：載入預測資料與模型

開啟系統後，依序點擊「載入模型」與「載入預測資料」，將所需檔案上傳至系統。請確保資料檔案為 CSV 格式。

步驟 3：選擇病房與床位

載入完畢後，選擇病房號碼，點選床位按鈕。系統右方資訊欄將顯示該床位的病人建議入住概率排序及推薦病人資料。

訓練模型步驟

步驟 1：打開資料夾

找到 SmartBed 資料夾。若無法執行，進入 src 子資料夾，點選 啟動程序.bat，系統將自動安裝所需環境。

步驟 2：載入訓練資料

系統中點選「載入訓練資料」，選擇包含訓練數據的 CSV 檔案，如簽床紀錄。

步驟 3：訓練模型

資料載入完成後，點選「訓練模型」。系統將自動處理數據並訓練模型，請耐心等待。訓練完成後，系統將顯示訓練結果。

步驟 4：調整程式碼

需調整模型邏輯時，開啟 src/model.py 檔案，直接修改訓練程式碼。
